eCPPTv3 Exam Preparation Guide

eCPPTv3 Exam Preparation Guide

eCPPTv3 (Certified Professional Penetration Tester) is a high-level certification that combines both web application penetration testing and network penetration testing. It typically requires around 2-5 months of dedicated study to pass.

Knowledge Required

1. Active Directory (AD) Attacks

A solid understanding of all AD attacks is essential:

  • AS-REP Roasting
  • DCSync Attack
  • Kerberoasting
  • Pass the Ticket
  • Unconstrained Delegation
  • Domain Trust Attacks

2. Web Application Attacks, Tools, and Frameworks

Proficiency in web application security, including:

  • Burp Suite
  • WordPress Security
  • Cross-Site Scripting (XSS)
  • SQL Injection
  • File Upload Vulnerabilities

3. Linux Privilege Escalation

A good knowledge of Linux and privilege escalation techniques:

  • Kernel Exploits
  • Passwords and File Permissions
  • Sudo Exploitation
  • SUID Exploitation
  • Capabilities Exploitation
  • Scheduled Tasks

4. Windows Privilege Escalation

Understanding Windows exploitation methods:

  • Kernel Exploits
  • Credential Exploitation
  • DLL Hijacking
  • Unquoted Service Paths
  • Token Manipulation
  • AlwaysInstallElevated

5. Exploitation Tools and Frameworks

Familiarity with various tools and frameworks:

  • PowerSploit
  • Metasploit
  • Burp Suite
  • WPScan
  • Hydra
  • SecLists
  • John the Ripper
  • Hashcat
